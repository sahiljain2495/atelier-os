# Atelier OS

A polished, installable, voice-first founder command room.

## Run locally
Open `index.html` in Chrome.

## Install on phone
The best experience is after hosting it online with HTTPS.
Then open the link on iPhone Safari and choose **Share → Add to Home Screen**.

## Voice commands
- Add under Menswear: call Anwar Ahmed for coat fabric
- Add under Bridal: follow up with Uttam Hazra PC
- Mark Anwar Ahmed done
- Delete hanger issue

## Current MVP features
- Luxury mobile-first UI
- Voice capture
- Text command fallback
- Auto-categorisation
- Waiting list
- Done list
- Daily log
- Export to clipboard
- Create Tomorrow rollover
- PWA manifest + app icons + service worker

## Important limitation
Your data is stored in the browser on that device. For a production version, the next step is Supabase login + cloud sync.
